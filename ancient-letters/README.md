# The Alchemist's Courier — Frontend Client

This directory contains the **React 18 + Vite + Tailwind CSS + Framer Motion** frontend for **The Alchemist's Courier**, an interactive Victorian-era correspondence and occult cipher application.

## 🚀 Quick Start

```bash
npm install
npm run dev
```

The application will launch on **http://localhost:5173/**.
